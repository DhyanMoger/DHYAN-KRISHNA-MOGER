This repository contains solutions for the Screening Test (4 Problems) in Python.

## Problem-1
Calculator using class to perform:
- Addition
- Subtraction
- Multiplication
- Division

## Problem-2

## Problem-3
Program to generate odd numbers:
- If input is odd → print odd numbers up to 'a'
- If input is even → print odd numbers up to 'a - 1'

## Problem-4
Count how many numbers in the list are multiples of 1 to 9.
